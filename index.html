Web Development

Using HTML, CSS, and JavaScript with pre-proceessors and build tools such as Sass and Grunt, I have a passion for developing pixel-perfect websites and apps while maintaining a semantic, modular, and DRY code base.

Responsive Mobile Design

"A user interface is like a joke. If you have to explain it, it's not that good." I strive to develop and implement responsive and aesthetically pleasing interfaces for websites and apps that adopt to any type of device, platform, or browser.

User Experience
